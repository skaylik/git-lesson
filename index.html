Это страница для зучения гита

1) команда, которую надо занть: git clone путь http к репозиторию

2) git status - статус текущего репозитория

3) git add - добавить файл под контроль гита

4) git commit -am "added new file index.html" - зафиксируем изменения

5) git push - отправляем файлы на сервер

5) git pull - кто угодно может скачать эти изменения
